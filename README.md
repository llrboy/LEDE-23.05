# OpenWrt x86_64

## 项目说明
- 自用的
- 项目使用 Github Actions 拉取 [Lean](https://github.com/coolsnowwolf/lede) 的 Openwrt 源码仓库进行云编译
- 固件默认管理地址：`192.168.81.1` 默认用户：`root` 默认密码：`password`
- 仅提供适配x86_64平台设备的 OpenWrt 固件
